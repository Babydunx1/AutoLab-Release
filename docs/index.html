Getting Started

🛠 Installation

ติดตั้งง่ายในไม่กี่ขั้นตอน พร้อมใช้งานทันที

🖥 Windows

1
กดปุ่ม Download Windows ด้านล่าง แล้วรอโหลดไฟล์ AutoLab_Setup.exe
2
ดับเบิลคลิกไฟล์ที่โหลดมา แล้วกด Install — ระบบจัดการให้อัตโนมัติ ไม่ต้องตั้งค่าอะไร
3
เปิด Adobe Premiere Pro ขึ้นมา (หรือรีสตาร์ทถ้าเปิดอยู่แล้ว)
4
ไปที่เมนูด้านบน Window › Extensions › AutoLab — แผง AutoLab จะเด้งขึ้นมา
5
สมัครทดลองใช้ด้วย email — จะได้รับสิทธิ์ใช้งานฟรี 14 วัน 🎉

🍎 macOS (Beta)

1
กดปุ่ม Download Mac ด้านล่าง แล้วรอโหลดไฟล์ AutoLab_Mac_Beta.zip
2
แตกไฟล์ zip แล้วทำตามคำแนะนำในโฟลเดอร์ที่แตกออกมา
3
เปิด Adobe Premiere Pro แล้วไปที่ Window › Extensions › AutoLab
⚠️ Mac เวอร์ชัน Beta — อาจพบปัญหาเรื่องสิทธิ์การรัน ให้ไปที่ System Settings › Privacy & Security แล้วกด Allow
